# Mock LWS Policy

This document is mock policy content for the Enterprise LWS GenAI Assistant PoC.
It is inspired by enterprise letter workflow patterns and contains no real PNC,
client, customer, account, or production data.

## Communication Controls

Letter generation requests must use approved mock template codes. Templates are
selected from customer segment, communication preference, product family, and
event reason. The workflow must not expose restricted account attributes in
letters unless the mock policy allows the section.

## Approval Rules

Disclosure language is controlled content. Any change to disclosure paragraphs
requires compliance approval, template versioning, and sample output review.

## Retention

Generated correspondence metadata is retained in the mock LWS database for audit
and troubleshooting. Printed output artifacts are represented only by synthetic
sample identifiers in this PoC.
